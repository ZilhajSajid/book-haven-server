## ğŸ’» Local Setup Guide

1. **Clone the repository**

   ```bash
   git clone https://github.com/ZilhajSajid/book-haven.git
   cd book-haven
   ```

2. **Install backend dependencies**

   ```bash
   cd server
   npm install
   ```

3. **Install frontend dependencies**

   ```bash
   cd ../client
   npm install
   ```

4. **Create `.env` files**

   * Backend: Add your MongoDB URI, JWT_SECRET, etc.
   * Frontend: Add Firebase config

5. **Run the server**

   ```bash
   cd server
   npm run dev
   ```

6. **Run the client**

   ```bash
   cd client
   npm start
   ```

7. **Open in browser**

   ```
   http://localhost:3000
   ```
